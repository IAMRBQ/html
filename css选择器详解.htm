css选择器详解

  • 选择器的种类

    • 符合选择器

      • 标签+类的形式
      • 作用：选中类名的标签元素
      • 示例：
        • div.box{padding:0px;}
        • 使类名为box的div元素的内边距为0px
      • 注意：支持多类条件

    • 后代选择器

      • 分隔符：空格
      • 作用：选择所有后代元素
      • 示例：
        • .box div{background:green;}
        • 效果：使所有类名为box的元素下的所有div元素的背景颜色为绿色
      • 注意：可以跨越多级

    • 子集选择器

      • 分隔符：>
      • 作用： 选择子代元素
      • 示例：
        • .box>div{font-size:0px;}
        • 效果：使所有类名为box的元素下的所有子代div元素的字体大小为0px
      • 注意：不能跨越多级

    • 群组选择器

      • 分隔符：,
      • 作用： 选择多种元素
      • 示例：
        • .box,div{width:100px;}
        • 效果：使所有类名为box的元素和所有div元素的宽度为100px
      • 注意：选择器之间并列关系

    • 毗邻选择器

      • 分隔符：+
      • 作用：选择紧相邻的下一个兄弟元素
      • 示例：
        • .box+div{margin:10px;}
        • 效果：使所有类名为box的相邻的下一个div元素的外边距为10px
      • 注意：只选中紧相邻的下一个，如果相邻元素不符合条件，则不生效

    • 多种选择器之间的复合

      • 示例：
        • div.head .left>div,.bottom+div{display:inline-block;}
        • 效果：使所有类名head下的所有类名left的所有子代的div和所有类名bottom的相邻的下一个div转换为行块元素

  • 选择器的优先级

    • 选择器类别

      • 默认样式<通配符选择器<标签选择器<类选择器<id选择器
      • *{margin:0px;}<div{margin:1px;}<.box{margin:2px;}<#box{marigin:3px;}

    • 选择器数量

      • 选择器类优先级相同时
      • #first{display:block;}<.box #first{display:inline-block;}

    • 执行顺序

      • 选择器类别和数量相同时
      • 靠后执行
      • 对内部样式的声明也同理

    • 样式的位置

      • 就近元素
      • 行内>头部>外部

    • !important

      • 直接声明最高优先级

    • 优先级的判断

      • !important
      • 行内>头部>外部
      • 判断选择器类优先级
      • 判断选择器数量
      • 判断执行顺序
